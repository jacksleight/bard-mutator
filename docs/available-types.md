# Available Types

Mutators support mutating the following node and mark types.

| Type                  | Tag Mutators | Data Mutators | Notes |
| --------------------- | :----------: | :-----------: | ----- |
| blockquote            | ●            | ●             |       |
| bullet_list           | ●            | ●             |       |
| code_block            | ●            | ●             |       |
| hard_break            | ●            | ●             |       |
| heading               | ●            | ●             |       |
| horizontal_rule       | ●            | ●             |       |
| image                 | ●            | ●             |       |
| list_item             | ●            | ●             |       |
| ordered_list          | ●            | ●             |       |
| paragraph             | ●            | ●             |       |
| table                 | ●            | ●             |       |
| table_cell            | ●            | ●             |       |
| table_header          | ●            | ●             |       |
| table_row             | ●            | ●             |       |
| bmu_root              |              | ●             | Internal node that wraps all content |
