# Available Types

Tag mutators support mutating the following node and mark types:

* **Nodes**
    * blockquote
    * bullet_list
    * code_block
    * hard_break
    * heading
    * horizontal_rule
    * image
    * list_item
    * ordered_list
    * paragraph
    * table
    * table_cell
    * table_header
    * table_row
* **Marks**
    * bold
    * code
    * italic
    * link
    * subscript
    * underline
    * strike
    * superscript
    * bts_span (part of [Bard Texstyle](https://github.com/jacksleight/statamic-bard-texstyle))
